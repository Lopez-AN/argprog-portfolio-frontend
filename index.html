Sitio del proyecto

Desarrolle el software de las placas y servidor para un sistema de máquinas expendedoras de productos de seguridad industrial basadas en Arduino y Raspberry-PI, comunicandose las mismas mediantes APIs con un servidor desarrollado en PHP, en el cual los clientes pueden consultar sus consumos, definir permisos, dar de alta productos y usuarios, etc.
Este proyecto tiene máquinas en las plantas de las principales empresas del pais como YPF, Grupo Exiros-Techint, Molinos Río de la Plata, BUNGE, y varios más.
